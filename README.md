# Laserchicken

Toolkit for handling point clouds created using airborne laser scanning (ALS). Find neighboring points in your point cloud and describe them as feature values. Included features:

 - point_density
 - echo_ratio
 - eigenv_1
 - eigenv_2
 - eigenv_3
 - z_entropy
 - max_z
 - min_z
 - mean_z
 - median_z
 - std_z
 - var_z
 - range
 - coeff_var_z
 - skew_z
 - kurto_z
 - normal_vector_1
 - normal_vector_2
 - normal_vector_3
 - slope
 - perc_10
 - perc_20
 - perc_30
 - perc_40
 - perc_50
 - perc_60
 - perc_70
 - perc_80
 - perc_90
 - perc_100
 - pulse_penetration_ratio
 - density_absolute_mean
 - sigma_z
